Weathery – A Weather Aggregator

February 02, 2011

As a sailing instructor it’s important to know as much information about the weather as possible. You need to be able to see how much wind there’s going to be, the height of the waves, the tides, if any storm is approaching, …

If you do this yourself, you usually have to check several websites. Each website has its advantages and disadvantages. Some are good in predicting the wind, others contain information about tides.

Weathery

Weathery is a web scraper that retrieves useful weather information from several sites. At this very moment it only gets the information for one specific place in Belgium, Koksijde.

Currently you can see the following:

  • Basic weather information from both Google and Wunderground.
  • Rainfall radar which is made by buienradar.nl. It allows you to see if any rain is coming.
  • A live-stream webcam from ‘t Zoetgenot.
  • Very good wind prediction.
  • UV emission prediction, sailing rating and wind information from meteovista.

The background image you’re seeing is randomly generated. It looks at the current weather condition (e.g. cloudy) adds a sea-like term (e.g. beach) and performs a search on flickr. If doesn’t find anything, a grey background is shown.

Hopefully some of you like it and maybe you’ll be able to use it the next time you jump on boat.
